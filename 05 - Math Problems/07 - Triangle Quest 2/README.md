### Pattern Calculation
```python
((10**i - 1) // 9) ** 2
```
Let's break down this calculation step-by-step:

1. **Calculate `10**i - 1`**:
   - `10**i` computes 10 raised to the power of `i`. For example, if `i = 3`, `10**i` is `1000`.
   - Subtracting `1` gives us a number consisting of `i` repetitions of `9`. For `i = 3`, `10**3 - 1` equals `999`.

2. **Divide by `9`**:
   - `((10**i - 1) // 9)` divides the result by `9`. This converts the number of `9`s into a number consisting of `i` repetitions of `1`. For `i = 3`, `999 // 9` equals `111`.

3. **Square the Result**:
   - `((10**i - 1) // 9) ** 2` squares the resulting number. For `i = 3`, `111 ** 2` equals `12321`.
